Privacy Policy

YouTube Comment Summarizer respects your privacy. This extension does not collect, store, or share any personally identifiable information (PII).

Data Collection

The extension reads publicly available YouTube video IDs to generate comment summaries. It optionally accepts user-provided Gemini API keys strictly for summarization purposes. These keys are stored locally in your browser and are not transmitted to any third parties.

Data Storage

Summaries generated are cached locally within the user's browser using Chrome's storage API. No user browsing history, personal data, or private content is collected or stored externally.

Contact

If you have any questions regarding this Privacy Policy, please contact us at your-email@example.com.
